# BCL Auction Platform — UI Kit

High-fidelity recreation of the Bromies Cricket League real-time auction product.
Composes the design-system primitives (`Button`, `Badge`, `Card`, `Avatar`,
`CountdownTimer`, `PlayerCard`, `TeamCard`, `BidPanel`) — it does not re-implement them.

## Screens
- **Home** (`Home.jsx`) — 7/5 hero, live hero auction card, team grid, featured players.
- **Auction Room** (`AuctionRoom.jsx`) — the main product. Theatre with player card,
  circular timer, bid panel, team bidding strip. Interactive: PLACE BID raises by the
  correct increment and resets the 20s timer; "SOLD" fires the gold celebration.
- **Team Dashboard** (`TeamDashboard.jsx`) — budget stats + squad list.
- **Player Database** (`PlayerDatabase.jsx`) — search + role/status filters + results.

## Run
Open `index.html`. It loads `_ds_bundle.js` (generated by the compiler), then `data.js`
(shared sample data on `window.BCL_DATA`), then each screen, then `app.jsx` (router).

Each screen file assigns its component to `window` so the Babel scripts can share scope.
