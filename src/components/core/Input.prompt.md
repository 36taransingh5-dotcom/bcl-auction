Cream text field used across the player database, admin panel and login. Ink focus ring, friendly rounded corners.

```jsx
<Input label="Search players" placeholder="Name, role…" iconLeft={<SearchIcon/>} />
<Input label="Base price" hint="In ₹ lakhs" size="lg" />
```

Sizes: sm / md / lg. Supports `label`, `hint`, `iconLeft`, and all native input props.
